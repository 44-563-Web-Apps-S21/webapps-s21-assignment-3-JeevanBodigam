How to use Postman

  1. Go to https://www.postman.com/
  2. Create an account if you don't have one
  3. Install the application and login with your created account
  4. Make sure you create a folder for a project
  5. Use appropraite GET,PUT,POST http verbs as per your request
  6. Run a simple GET request for testing.
  7. Check the response and response code are correct
  8. Move to other request
